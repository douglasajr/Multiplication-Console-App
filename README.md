# Multiplication CLI App

Esta es una aplicación de consola construida en Node.js que permite generar tablas de multiplicación utilizando flags (argumentos) desde la terminal.

El usuario puede proporcionar:
	•	-b o --base → El número base de la tabla
	•	-l o --limit → El límite hasta donde se multiplicará la base
	•	-n o --name -> El nombre del archivo a crear(se recomienda dejarlo prederteminado)
	•	-d o --directory -> El nombre del carpeta a crear donde se va guarda el archivo


Una vez ejecutado el comando, la aplicación genera un archivo .txt con la tabla de multiplicación generada y lo guarda automáticamente.
